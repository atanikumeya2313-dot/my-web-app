# 積み立てシミュレーター

毎月の積み立て額・年利・NISA設定を入力すると、将来の資産推移をリアルタイムでシミュレーションするブラウザアプリ。

**本番URL**: https://savings-sim.vercel.app

---

## 機能一覧

### 入力パラメーター（スライダーでリアルタイム反映）

| 項目 | 範囲 | デフォルト |
|------|------|------------|
| 毎月の積み立て額 | 1,000〜100,000円 | 30,000円 |
| 年利（想定リターン） | 0.1〜15% | 5.0% |
| 積み立て期間 | 1〜40年 | 20年 |
| 初期投資額 | 0円〜 | 0円 |

### NISA設定

| 選択肢 | 年間非課税枠 | 生涯上限 |
|--------|------------|----------|
| 使わない | なし | — |
| つみたて投資枠 | 120万円/年 | 1,800万円 |
| 成長投資枠 | 240万円/年 | 1,800万円 |

NISA枠を超えた積み立て分は課税口座扱い（利益に20.315%課税）として自動計算されます。

### 結果表示

| 項目 | 説明 |
|------|------|
| 税引後 最終残高 | NISA非課税分＋課税後利益を合算した手取り額 |
| NISA非課税益 | NISA口座内の運用益（非課税） |
| NISA節税額 | NISAを使わなかった場合と比べた節税額 |
| 年次推移グラフ | 元本・NISA非課税益・課税後利益の積み上げ棒グラフ |
| NISA生涯枠の消化 | 1,800万円上限に対する利用率バー |
| 年次詳細テーブル | 各年の元本・残高・NISA益・税引後残高（開閉式） |

---

## 計算仕様

- **複利計算**: 月次複利（年利を月利に換算して毎月適用）
- **NISA年間枠**: 毎年1月にリセット。枠を超えた分は自動的に課税口座へ
- **NISA生涯上限**: 1,800万円到達後は全額課税口座扱い
- **税率**: 20.315%（所得税15%＋住民税5%＋復興特別所得税0.315%）
- **税引後残高の計算式**:
  ```
  NISA残高（元本＋非課税益）
  ＋ 課税口座元本
  ＋ 課税口座利益 × (1 - 0.20315)
  ```

## 開発・デプロイ

```powershell
# savings-sim/ ディレクトリで実行

# 開発サーバー起動
npm run dev
# → http://localhost:3000

# Vercel へデプロイ
vercel --prod
```
